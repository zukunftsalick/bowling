# Tenpin Bowling Scoring

Calculates and prints ten pin bowling scores.

#### Requirements:
* Ruby 2.2.4
* rspec

#### Installation:
````
$ bundle install
`````

#### Usage:

```
$ ruby bowling_game.rb 1, 2, 3, 4, 5, 5
 20
```

#### Running Tests:

```
$ rspec .
```
